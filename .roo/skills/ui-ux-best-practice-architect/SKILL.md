name: ui-ux-best-practice-architect
description: Setiap kali membangun komponen UI, pastikan:

Accessibility (WCAG): Rasio kontras teks minimal 4.5:1. Gunakan elemen semantik (<nav>, <main>, <section>).

Design Tokens: Gunakan variabel Tailwind untuk warna dan spacing (jangan hardcoded).

State Management: Selalu sertakan visual state untuk Hover, Active, Disabled, dan Loading (Skeleton).

Dynamic Data: Gunakan library faker-js atau buat objek JSON mock yang kompleks untuk mensimulasikan data nyata agar stakeholder mendapat gambaran akurat.
